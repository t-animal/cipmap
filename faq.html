Probably frequently asked questions

Where do you get your data from?

I use the public information which every multi user system provides. They are collected by a programm called cipmap server that prints it on faui02:1338.

What does "User: N/A" mean?

This message means that there is a user logged in who has not opted in to displaying his username on the site. See below for information how to opt out globally.

What do the times mean?

The server tells me when it last reached the host or when it last checked wether a user was logged in. I round this time to minutes and show it on the website. Displaying seconds was too unresting for my taste ;).

What do the colors mean?

Duh. Green means the pc/workstation is unoccupied, red means it is not. Gray means there are currently no information for this machine.

Can I access the site from outside the university?

Sure. All you need is a VPN connection to the university network. If the CIP admins are OK with it I might offer a password-protected version of this site outside the university network in the future.

How can I make my username show up on the website?

Please try the opt-in system. No promises that everything works already, but go ahead and give it a try.

I don't want my name to show up there or anywhere else!

When it comes to this website you do not have to do anything: it is an opt-in system, which means you only have to act if you want your name to show up. All names are filtered before they are sent to the users.
I'm not responsible for the cipmap server. However, you can opt-out from the data collection by touching a file with your username in /proj/cipmap: $> touch /proj/cipmap/`whoami`. Please note that even though you may have opted out from the cipmap, there is still plenty of information available for someone to find out if and where you are logged in. There is no way to prevent it.
